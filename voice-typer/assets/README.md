# Voice Typer Assets

This directory holds Voice Typer assets for the OpenVibeAI-data repository.

- `generated/` — **automation-owned**. Synced from
  [`whisper-vibes`](https://github.com/mariuszRep/whisper-vibes) via GitHub
  Action pull requests. Do not edit manually.
- Other subdirectories (e.g. `manual/`) — for manually maintained assets, if
  needed in the future.
